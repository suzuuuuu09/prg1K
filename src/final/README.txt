## 概要
これは「Termiplay」というターミナルで動画を再生するツールです。
これを作った経緯としては東方projectの「Bad Apple」という楽曲があり、そのMVを本来なら動画を流せない場所で動画を流すというミームから着想を得て、ターミナルでも再生できるようにしてみようと考えたためです。
もともとはPythonで書かれていたものをC言語でも実行できるように移植しました。

## 実行方法
VSCodeなどの内蔵ターミナルではなく、「ターミナル」や「wezterm」のようなアプリで実行する必要があります。
また、実行するときは全画面で行う必要があります。全画面で実行しないとレイアウトが崩れてうまく動画が表示できない可能性があります。

1. パッケージ(opencv)のインストール
以下のコマンドを実行する。
```zsh
brew install opencv
```
2. `run.zsh`を実行
