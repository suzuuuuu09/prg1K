# ターミナル動画プレイヤー (C言語版)

このプログラムは、ターミナル上で動画を再生するC言語で書かれたプレイヤーです。
ffmpegを使用して動画を処理し、ANSIカラーコードでカラフルな出力を実現しています。

## 必要な環境

- macOS/Linux
- gcc コンパイラ
- ffmpeg (インストール済み)

## ビルド方法

```bash
make
```

または

```bash
gcc -Wall -Wextra -std=c99 -o video_player 15_k25012_issues.c
```

## 使用方法

1. プログラムを実行:
   ```bash
   ./video_player
   ```

2. 動画ファイルのパスを入力してください
   例: `video/badapple.mp4`

3. 動画が全画面で再生されます
   - 'q' または 'Q' を押すと再生を終了できます

## 機能

- フルカラーでの動画再生
- 全画面表示
- アニメーション付きロゴ表示
- ローディングアニメーション
- キーボード入力による再生制御

## 対応フォーマット

ffmpegが対応している全ての動画フォーマット:
- MP4
- AVI
- MOV
- MKV
- WebM
- など

## 注意事項

- 動画の解像度はターミナルのサイズに自動調整されます
- 30FPSで再生されます
- 一時ファイルは `/tmp/video_frames.raw` に作成され、再生後に自動削除されます
- Ctrl+C で強制終了した場合、ターミナルの設定が正常に戻ります

## クリーンアップ

```bash
make clean
```

## トラブルシューティング

### 動画が再生されない場合
- ffmpegが正しくインストールされているか確認
- 動画ファイルのパスが正しいか確認
- ファイルの権限を確認

### 表示が乱れる場合
- ターミナルがTrueColorをサポートしているか確認
- ターミナルのサイズを調整してみてください
